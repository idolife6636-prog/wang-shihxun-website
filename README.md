## 🚀 快速開始

### 本地測試
```bash
# 使用 Python 啟動本地伺服器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然後在瀏覽器打開 `http://localhost:8000`

### 部署到 Netlify

1. 連接你的 GitHub 儲存庫到 Netlify
2. 自動部署完成
3. 訪問生成的 URL
